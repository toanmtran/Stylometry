# Neural Network — Stylometric Author Classification

MLP-based authorship classifier trained on stylometric features extracted from LessWrong blog posts. Five target authors: Eliezer Yudkowsky, Johnswentworth, Raemon, Scottalexander, and Zvi.

---

## Evaluation Design

Data is split **60 / 20 / 20** (train / dev / test), stratified by author. All 25 combinations of feature subset × network depth are trained on the train split and ranked by dev accuracy. The best combination is retrained on train+dev, then evaluated **once** on the held-out test set. The two cases (with/without outliers) are run independently.

```
train (60%) ─► fit all 25 configs ─► rank by dev acc ─► pick best
dev   (20%) ─► score all 25 configs ─────────────────────────────┘
                                                          │
train+dev ──────────────────────────── retrain best ──────┘
test  (20%) ────────────────────────── final report (touched once)
```

---

## Quick Start

```bash
# Run both cases (with and without outliers) — writes two result files
python neural_network_code.py

# Same, with early stopping enabled during training
python neural_network_code.py --early-stopping

# Custom input file
python neural_network_code.py --input my_features.csv

# Standalone outlier removal only
python outlier_removal.py
```

## Files

### Python Scripts

#### `neural_network_code.py`
Main training and evaluation script. Always runs both cases (with and without outliers) and writes one Markdown result file per case.

| Flag | Default | Description |
|------|---------|-------------|
| `--input` | `author_features_extracted_full.csv` | Input feature CSV |
| `--early-stopping` | off | Enable early stopping in MLPClassifier |

**Outputs:** `results_with_outliers.md`, `results_without_outliers.md`

#### `outlier_removal.py`
Standalone script that applies per-author Isolation Forest (`contamination="auto"`) to `author_features_extracted_full.csv` and writes the cleaned result to `feature_extracted_without_outliers.csv`. Prints a per-author removal count.

---

### Datasets

#### `author_features_extracted_full.csv`
Full feature matrix: 107 stylometric features per passage, all authors, all passages (outliers included). Each row is one passage keyed by `author` and `passage_id`.

| Category | Examples |
|----------|---------|
| Lexical richness | `hapax_ratio`, `yule_k`, `simpson_d`, `brunet_w`, `honore_r` |
| Word-length stats | `avg_word_len`, `std_word_len`, `word_len_1_frac` … `word_len_6_frac` |
| Sentence stats | `avg_sent_len`, `std_sent_len`, `median_sent_len`, `max_sent_len` |
| Function words | `fw_the`, `fw_be`, `fw_to` … (top-50 English function words) |
| Punctuation rates | `punct_comma_rate`, `punct_semicolon_rate`, `punct_dash_rate` … |
| Category rates | `cat_hedge_rate`, `cat_amplifier_rate`, `cat_discourse_rate` |
| POS proportions | `pos_noun`, `pos_verb`, `pos_adj`, `pos_adv`, `pos_modal` … |
| Surface features | `uppercase_ratio`, `contraction_rate`, `pronoun_start_rate` |
| Character rates | `char_a_rate`, `char_e_rate`, `char_i_rate`, `char_o_rate`, `char_u_rate` |

#### `feature_extracted_without_outliers.csv`
Same structure as above with per-author outlier passages removed by Isolation Forest (723 → 686 rows). Generated by `outlier_removal.py`.

### Evaluation Results

#### `results_with_outliers.md`
Dev/test evaluation on the **full dataset** (723 passages). Best model selected by dev accuracy, retrained on train+dev, evaluated on test.

- Best config (dev): **Top 74 features · Depth 1 (64,)** — Dev: 94.48%
- **Test accuracy: 93.79%**

#### `results_without_outliers.md`
Dev/test evaluation on the **outlier-removed dataset** (686 passages). Same procedure.

- Best config (dev): **Top 50 features · Depth 1 (64,)** — Dev: 93.43%
- **Test accuracy: 96.38%**

#### `evaluation_results_full_and_subsets (1).md`
Legacy: 5-fold CV results on the full dataset across all configs (no train/dev/test split).

#### `Results_without_outlier.md`
Legacy: 5-fold CV results on the outlier-removed dataset.

#### `with_early_stopping.md`
Legacy: 5-fold CV with early stopping enabled.

## Model Architecture

| Setting | Value |
|---------|-------|
| Type | MLP (fully connected feedforward) |
| Hidden activation | ReLU |
| Output activation | Softmax (5 classes) |
| Weight init | Xavier uniform |
| Optimizer | Adam |
| Loss | Cross-entropy |
| Preprocessing | StandardScaler (fit on train split only) |
| Model selection | Dev set (20% held-out) |
| Final evaluation | Test set (20% held-out, touched once) |

---

## Summary of Results

| Case | Best Config (by dev) | Dev Acc | Test Acc |
|------|----------------------|---------|----------|
| With outliers    | Top 74 features · Depth 1 (64,) | 94.48% | 93.79% |
| Without outliers | Top 50 features · Depth 1 (64,) | 93.43% | **96.38%** |

Shallow networks (depth 1–2) consistently win. Depth ≥ 10 degrades accuracy; depth 50 collapses entirely (~21%, majority-class prediction). Outlier removal yields the best test accuracy.
